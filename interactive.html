07 Interactive

Gender:

In this example:

1. There's a text input field for entering a name.

2. A checkbox for subscribing to a newsletter.

3. Radio buttons for selecting gender.

4. A select dropdown for selecting a country.

5. A button for submitting the form, which triggers the submitForm() JavaScript function when clicked. The function currently displays an alert, but you can modify it to perform other actions like sending form data to a server.



html

<!-- Text Input Field -->
<div class="c-int-element l-pb">
<label for="name">Name:</label>
<input type="text" id="name" name="name" placeholder="Enter your name">
</div>

<!-- Checkbox -->
<div class="c-int-element l-pb">
<input type="checkbox" id="subscribe" name="subscribe">
<label for="subscribe">Subscribe to newsletter</label>
</div>  

<!-- Radio Buttons -->
<div class="c-int-element l-pb">
<fieldset>
<legend>Gender:</legend>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label><br>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label><br>
<input type="radio" id="other" name="gender" value="other">
<label for="other">Other</label>
</fieldset>
</div>

<!-- Select Dropdown -->
<div class="c-int-element l-pb">
<label for="country">Country:</label>
<select id="country" name="country">
<option value="usa">USA</option>
<option value="canada">Canada</option>
<option value="uk">UK</option>
<option value="australia">Australia</option>
</select>
</div>

<!-- Button -->
<div class="c-int-element l-pb">
<button class="button" type="button" onclick="submitForm()">Submit</button>
</div>



CSS
