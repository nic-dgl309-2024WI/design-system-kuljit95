05 Containers

Flex and Grid Containers

Class Name
Description
.l-flex
Setting display flex and flex direction will be row wise.
.l-grid
Setting display grid and flex direction will be column wise.

Full width and Max width Containers

Class Name
Description
.full-width
Making content width to 100%.
.max-width-desktop
Setting width to 1200px for desktop.
.max-width-mobile
Setting width to 100% for mobile.
.width-700
Setting max width to 700px.
